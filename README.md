# izumi works

## 概要

プログラミングスクールtech.isのカリキュラムに沿って作成したcodeをまとめてます。

## フォルダ構成

│  
┌─ `bbs_ci`  
│   // codeigniterを使ったひことこ掲示板 - html5,css3,php,sql  
├─ `calender_year`  
│   // 内閣府が定める祝日入りの年間カレンダー - html5,css3,php,sql  
├─ `crm_adminlte_ci`  
│   // codeigniter・adminlte3を使った顧客管理システム - html5,css3,php,sql,bootstrap4  
├─ `form_PDO`  
│   // PDOを使った会員登録・ログインシステム - html5,css3,php,sql  
├─ `login_ci`  
│   // codeigniterを使った会員登録・ログインシステム - html5,css3,php,sql  
├─ `message_board`  
│   // phpのひとこと掲示板 - html5,css3,php,sql  
└─ `update_resize`  
     // phpの画像アップロード・リサイズ可能 - html5,css3,php  

## 導入方法

apacheのDocumentRootに`git clone`するかzipを展開してください。
